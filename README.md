# angular101
Este proyecto contiene 10 ejemplos muy sencillos para conocer AngularJS

* **ej1:** ng-app primera aplicación
* **ej2:** ejecutar expresiones y ng-init
* **ej3:** Databinding angular vs jquery
* **ej4:** ng-model
* **ej5:** ng-repeat
* **ej6:** ng-show/ng-hide
* **ej7:** Filtros
* **ej8:** ng-app módulos
* **ej9:** Módulos, controladores y un poco de orden
* **ej10:** Automatización y aplicación NodeJS + AngularJS (aplicación basada en HotTowel Angular)

## Pre-requisitos

1. Instalar [Node.js](http://nodejs.org)
 - OSX [homebrew](http://brew.sh) `brew install node`
 - Windows [chocolatey](https://chocolatey.org/) `choco install nodejs`

2. Instalar bower global

    ```bash
    npm install -g bower gulp nodemon
    ```

### Instalar dependencias

Situarse en la carpeta ej10 y ejecutar los siguientes comandos para que se descarguen e instalen las dependencias necesarias del proyecto.

 - `npm install`
 - `bower install`

El resto de instrucciones se pueden consultar en el README.md del directorio ej10
